# Blender_bulk_export_root_node_objects
This script allows you to batch export multiple models in Blender, automatically setting each model's origin to the center — optimized for game engine usage.

Blender Version : 3.6

Steps: Goto scripting tab -> new script -> paste the script and run

Check folder: E:/Users/YourName/Desktop/ExportedGLB/

Works with folder structure:

```text
RootNode
└── Interiors
    ├── Armchair_1
    │   └── (mesh inside)
    ├── Armchair_2
    │   └── (mesh inside)
    └── Armchair_3 ...
